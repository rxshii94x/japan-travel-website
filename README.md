## 🚀 Installation

Clone the repository:

```bash
git clone https://github.com/rxshii94x/japan-travel-website.git
```

Move into project:

```bash
cd japan-travel-website/app
```

Install dependencies:

```bash
npm install
```

Run development server:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```
